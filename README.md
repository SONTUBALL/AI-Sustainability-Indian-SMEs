Project Name: The Driving Factors for Integrating AI in the 
Sustainable Development of Indian SMEs

Project Overview:
This project analyzes the driving factors for integrating Artificial Intelligence (AI) in the sustainable development of Indian SMEs using data analytics and machine learning techniques.

Objectives:
1. Predict AI optimization adoption
2. Cluster SMEs based on sustainability factors
3. Compare AI vs Non-AI SMEs
4. Evaluate model performance using Confusion Matrix

Tools and Technologies: Python, SQL, Power BI, Excel, and Machine Learning (Supervised and Unsupervised Learning)
  
Dataset: SME sustainability dataset.

Key Insights:
AI adoption improves sustainability efficiency
